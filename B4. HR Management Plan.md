## 3. ROLES & TANGGUNG JAWAB

### 3.1 Tim Manajemen

#### Project Manager (PM)

**Kualifikasi (KAK hal. 12)**:

- Minimal S1 Teknik Informatika/Sistem Informasi
- **Pengalaman**: 6+ tahun sebagai PM aplikasi berbasis web (pemerintah/swasta)
- Bukti: CV, KTP, NPWP
- **Sertifikasi**: PMP/PRINCE2 (preferred)
- **Track Record**: Min 5 aplikasi berbasis web selesai

**Tanggung Jawab**: Overall project success, stakeholder management (PPK, Direktur, Users), budget & schedule control, risk & issue management, team leadership, reporting (weekly, monthly), change control, quality assurance.

**Alokasi Waktu**: Full-time 6 bulan

#### Assistant Project Manager

**Kualifikasi**: S1 Teknik Informatika/Manajemen, 3+ tahun experience PM support.

**Tanggung Jawab**: Administrative support untuk PM, meeting coordination, documentation management, progress tracking & reporting, resource scheduling, communication facilitation.

**Alokasi Waktu**: Full-time 6 bulan

### 3.2 Tim Teknis

#### System Analyst (SA)

**Kualifikasi (KAK hal. 13)**:

- Minimal S1 Teknik Informatika/Sistem Informasi
- **Pengalaman**: 4+ tahun sebagai SA (design arsitektur, alur aplikasi, basis data, tampilan)
- Bukti pernah develop aplikasi berbasis web min 4 aplikasi
- CV, KTP, NPWP

**Tanggung Jawab**: Requirements analysis & validation, system design (architecture, database, UI/UX flow), technical leadership, code review, QA Lead (dual role), technical documentation, integration planning (SIMPONI).

**Alokasi Waktu**: Full-time 6 bulan

#### Senior Programmer 1, 2, 3 (SP1-3)

**Kualifikasi (KAK hal. 14)**:

- Minimal S1 Teknik Informatika
- **Pengalaman**: 5+ tahun sebagai programmer aplikasi berbasis web
- **Skills**: PHP, JavaScript, React/Vue, MySQL/PostgreSQL
- **Expertise**: Backend development, API integration, security implementation

**Tanggung Jawab**: Coding (backend + frontend), unit testing, code review (peer), integration dengan SIMPONI, QR Digital Signature implementation, CAPTCHA integration, performance optimization.

**Alokasi Waktu**:

- SP1: Full-time 6 bulan
- SP2: Full-time 6 bulan
- SP3: Full-time 4 bulan (focused on core dev phase)

#### Database Administrator (DBA)

**Kualifikasi**: S1 Teknik Informatika/Database specialization, 3+ tahun DBA experience. Skills: SQL Server 2016, database design, optimization, backup/recovery.

**Tanggung Jawab**: Database design & ERD, database implementation & optimization, data migration (dari KANMIS 2.1), backup & recovery strategy, performance tuning, security (access control, encryption).

**Alokasi Waktu**: 4 bulan (Bulan 1-4, overlap dengan development)

#### UI/UX Designer

**Kualifikasi**: S1 Design Komunikasi Visual/Informatika, 3+ tahun UI/UX experience web applications. Skills: Figma, Adobe XD, responsive design.

**Tanggung Jawab**: UI/UX mockup design, user flow diagrams, responsive design specifications, style guide creation, usability testing support, design review & iteration.

**Alokasi Waktu**: 3 bulan (Bulan 1-3, design phase)

### 3.3 Tim QA & Documentation

#### QA/Tester

**Kualifikasi**: S1 Teknik Informatika, 2+ tahun QA/testing experience. Skills: Test automation, Selenium, JMeter, manual testing.

**Tanggung Jawab**: Test plan creation, test case development, test execution (functional, integration, system), defect logging & tracking, UAT coordination, regression testing.

**Alokasi Waktu**: 2 bulan (Bulan 5-6, testing & UAT phase)

#### Security Tester

**Kualifikasi**: S1 Teknik Informatika/Cybersecurity, 3+ tahun security testing experience. Skills: Penetration testing, OWASP, vulnerability assessment. Certifications: CEH/OSCP (preferred).

**Tanggung Jawab**: Security testing plan, penetration testing, vulnerability scanning, CAPTCHA & QR Signature validation, security audit, remediation verification.

**Alokasi Waktu**: 0,5 bulan (15 hari, focused security sprint)

#### Technical Writer

**Kualifikasi**: S1 Teknik Informatika/Sastra, 2+ tahun technical writing experience.

**Tanggung Jawab**: Technical documentation (architecture, API, deployment), user manual (bahasa Indonesia), admin guide, training materials, FAQ documentation.

**Alokasi Waktu**: 2 bulan (Bulan 5-6, parallel dengan testing)

## 7. PERFORMANCE MANAGEMENT

### 7.1 Performance Metrics

**Individual Metrics**: Task completion rate vs planned, code quality (review feedback, defect density), collaboration effectiveness (peer feedback), innovation & problem-solving.

**Team Metrics**: Sprint velocity (story points completed), defect escape rate, on-time delivery percentage.

### 7.2 Performance Reviews

**Weekly 1-on-1**: PM dengan setiap team member - Progress check, blocker removal, feedback exchange.

**Monthly Performance Review**: Formal assessment, goal setting, development planning.

**End-of-Project Review**: Overall contribution assessment, lessons learned, reference letter (untuk high performers).

---

## 8. RECOGNITION & REWARDS

### 8.1 Program Recognition

**Individual Recognition**: "Developer of the Month" (based on peer voting + PM), "Bug Hunter Award" (most critical bugs found), "Innovation Award" (best technical solution).

**Team Recognition**: Milestone celebrations (M2, M3, M4, M5), project completion bonus (kalau on-time, on-budget).

### 8.2 Rewards

**Financial**: Performance bonus (10% of salary untuk top performers), milestone bonuses (aligned dengan payment terms).

**Non-Financial**: Certificates of appreciation, LinkedIn recommendations, professional development opportunities.

## 9. COMMUNICATION PLAN UNTUK HR

### 9.1 Team Communication

**Daily Standup**: 15 min, Mon-Fri - Kemarin ngapain, hari ini mau ngapain, ada blocker apa.

**Weekly Team Meeting**: 1 jam, Jumat - Sprint review, next week planning, Q&A.

**Monthly All-Hands**: 2 jam - Project status update, stakeholder feedback, team achievements.

### 9.2 One-on-One Communication

PM ↔ Team Members: Weekly SA ↔ Developers: As needed (design discussions) QA ↔ Developers: Daily (selama testing phase)

---

## 10. CONFLICT MANAGEMENT

### 10.1 Conflict Resolution Process

**Level 1**: Direct discussion (team members resolve sendiri) **Level 2**: PM mediation **Level 3**: Escalation ke PPK (kalau unresolved)

**Conflict Prevention**: Clear roles & responsibilities (RACI), regular communication, early issue identification, team building activities.

### 10.2 Common Conflicts & Solutions

**Conflict**: Resource allocation disputes **Solution**: Priority matrix, PM arbitration

**Conflict**: Technical disagreements **Solution**: Architecture review board decision

**Conflict**: Workload imbalance **Solution**: Re-distribution, additional resources kalau perlu

---

## 11. SAFETY & WELLBEING

### 11.1 Work-Life Balance

**Working Hours**: 8am-5pm, Mon-Fri (flexible) **Overtime Policy**: Minimize overtime, kompensasi kalau needed **Remote Work**: Allowed 2 hari/minggu (setelah Bulan 1)

### 11.2 Health & Safety

**Workspace**: Ergonomic setup, proper lighting **Breaks**: Regular breaks encouraged (Pomodoro technique) **Mental Health**: Open-door policy, support resources
