Dokumen ini mengatur bagaimana biaya proyek akan kita kelola - mulai dari estimasi, budgeting, sampai monitoring dan control. Total nilai kontrak adalah **Rp 919 juta** untuk durasi **150 hari kerja** (sesuai KAK BSN halaman 3).

## 2. PENDEKATAN MANAJEMEN BIAYA

### 2.1 Filosofi Manajemen Biaya

**Tujuan Utama**: Selesaikan proyek dalam budget Rp 919 juta sambil maintain standar kualitas yang diharapkan.

**Prinsip Kunci:**

1. **Transparansi**: Semua pengeluaran terdokumentasi dan bisa dilacak
2. **Akurasi**: Estimasi based on market rates + data historis
3. **Kontrol**: Variance threshold maksimal ±10%
4. **Value for Stakeholder**: Optimasi value yang delivered per rupiah

### 2.2 Proses Manajemen Biaya

Mengikuti PMBOK, kita jalankan:

1. **Plan Cost Management** → Dokumen yang sedang Anda baca
2. **Estimate Costs** → Bottom-up estimation dari WBS
3. **Determine Budget** → Aggregation + time-phasing
4. **Control Costs** → EVM tracking + variance analysis

## 4. METODOLOGI ESTIMASI BIAYA

### 4.1 Teknik Estimasi yang Dipakai

**Bottom-Up Estimating**: Kita breakdown dari work packages WBS → individual cost estimates → aggregate ke summary level. Ini metode paling akurat menurut Schwalbe (2019, hal 274).

**Analogous Estimating**: Pakai reference dari:

- SIMPONI Integration project (Kemenkeu) - Rp 850 juta
- SIKAP (KAN) - Rp 720 juta
- Adjustment: +15% untuk complexity KANMIS (40+ fitur)

**Parametric Estimating**: Formula: Total LOC (Lines of Code) × Cost per LOC Estimasi: 50.000 LOC × Rp 18.000/LOC = Rp 900 juta (validation check)

### 4.2 Basis Estimasi

**Labor Costs**: Sumber data:

- Market survey: JobStreet, Glints, Indeed (Q2 2024)
- Company internal benchmarks
- Asosiasi IT Indonesia salary guide 2024

**Infrastructure Costs**:

- Vendor quotes: AWS, Azure, Google Cloud
- Pilihan final: AWS (best price-performance ratio)

**Operational Costs**:

- Data historis dari proyek sejenis
- Inflation adjustment +5% (2024 vs 2023)

## 6. EARNED VALUE MANAGEMENT (EVM)

### 6.1 Framework EVM

**Budget at Completion (BAC)**: Rp 919 juta

**Metrik Utama EVM** (Schwalbe 2019, hal 283-285):

|Metrik|Formula|Interpretasi|
|---|---|---|
|**Planned Value (PV)**|Budget yang authorized untuk scheduled work|Baseline|
|**Earned Value (EV)**|Budget untuk work yang sudah performed|Progress aktual|
|**Actual Cost (AC)**|Realized cost untuk work performed|Spending aktual|
|**Cost Variance (CV)**|EV - AC|Negatif = over budget|
|**Schedule Variance (SV)**|EV - PV|Negatif = terlambat|
|**Cost Performance Index (CPI)**|EV / AC|< 1.0 = over budget|
|**Schedule Performance Index (SPI)**|EV / PV|< 1.0 = terlambat|
|**Estimate at Completion (EAC)**|BAC / CPI|Proyeksi total cost|
|**Estimate to Complete (ETC)**|EAC - AC|Sisa budget yang dibutuhkan|
|**Variance at Completion (VAC)**|BAC - EAC|Proyeksi over/under|

Schwalbe (2019, hal 284): "Cost variance adalah earned value dikurangi actual cost. CV negatif berarti pekerjaan menghabiskan biaya lebih dari rencana."

### 6.2 Monthly Planned Value (PV) Baseline

|Bulan|% Complete|PV (Kumulatif)|
|---|---|---|
|1|10%|Rp 91,9 juta|
|2|25%|Rp 229,75 juta|
|3|45%|Rp 413,55 juta|
|4|65%|Rp 597,35 juta|
|5|85%|Rp 781,15 juta|
|6|100%|Rp 919 juta|

### 6.3 Contoh Kalkulasi EVM (Bulan 3)

**Data Skenario**:

- AC (Actual Cost) = Rp 430 juta
- % Complete = 40%
- EV = 40% × Rp 919 juta = Rp 367,6 juta
- PV = Rp 413,55 juta

**Kalkulasi**:

1. **SV** = EV - PV = 367,6 juta - 413,55 juta = **-Rp 45,95 juta** (terlambat)
2. **CV** = EV - AC = 367,6 juta - 430 juta = **-Rp 62,4 juta** (over budget)
3. **SPI** = EV / PV = 367,6 / 413,55 = **0,89** (11% behind schedule)
4. **CPI** = EV / AC = 367,6 / 430 = **0,85** (15% over budget)
5. **EAC** = BAC / CPI = 919 juta / 0,85 = **Rp 1.081 juta** (projected overrun)
6. **ETC** = EAC - AC = 1.081 - 430 = **Rp 651 juta** (additional needed)
7. **VAC** = BAC - EAC = 919 - 1.081 = **-Rp 162 juta** (projected overrun)

**Interpretasi**: Proyek over budget DAN behind schedule → butuh **corrective action segera**.

## 7. PROSEDUR KONTROL BIAYA

### 7.1 Performance Thresholds

**Traffic Light System**:

- **Hijau** (CPI ≥ 0,95): On track, lanjutkan
- **Kuning** (0,85 ≤ CPI < 0,95): Hati-hati, review & optimize
- **Merah** (CPI < 0,85): Critical, corrective action wajib

### 7.2 Monitoring & Reporting

**Weekly Cost Tracking**:

- Update AC setiap Jumat
- Hitung EV based on % complete (verified PM + SA)
- Variance analysis
- Update forecast EAC

**Monthly Executive Report**:

- EVM dashboard (dalam bentuk grafik)
- Trend analysis (CPI, SPI trends)
- Budget forecast
- Rekomendasi

**Real-Time Dashboard**: Tools: Excel Power BI / Google Sheets Metrics: AC, EV, PV, CPI, SPI, EAC Update: Harian

### 7.3 Change Control Process

**Untuk Cost Changes > Rp 10 juta**:

1. Submit Change Request ke PM
2. Impact analysis (cost, schedule, scope)
3. Review oleh CCB (Change Control Board)
4. Kalau approved → update budget baseline
5. Butuh approval formal dari PPK (Habib Nurhasan)

**Kriteria Budget Rebaseline**:

- Cumulative changes > 10% BAC
- Butuh approval Direktur BSN
- Full documentation & justification

## 9. ANALISIS FINANSIAL

### 9.1 Cost-Benefit Analysis

**Investasi**: Rp 919 juta

**Expected Benefits** (dari BSN Business Case):

- Efisiensi operasional: Rp 300 juta/tahun
- Reduced manual processing: Rp 150 juta/tahun
- Improved compliance: Rp 100 juta/tahun (risk avoidance)

**Total Annual Benefit**: Rp 300 juta (estimasi konservatif)

### 9.2 Net Present Value (NPV)

**Asumsi**:

- Discount rate: 10% per tahun
- Analysis period: 4 tahun
- Annual benefit: Rp 300 juta

**Kalkulasi**:

- Tahun 0: -Rp 919 juta (investasi)
- Tahun 1: Rp 300 juta / 1,1¹ = Rp 272,73 juta
- Tahun 2: Rp 300 juta / 1,1² = Rp 247,93 juta
- Tahun 3: Rp 300 juta / 1,1³ = Rp 225,39 juta
- Tahun 4: Rp 300 juta / 1,1⁴ = Rp 204,90 juta

**NPV** = -919 juta + 950,95 juta = **+Rp 31,95 juta** ✓ (Positive = viable)

### 9.3 Return on Investment (ROI)

**ROI** = (Total Benefit - Total Cost) / Total Cost × 100% **ROI** = (1.200 juta - 919 juta) / 919 juta × 100% = **30,6%**

**Payback Period** = Investasi / Annual Benefit = 919 juta / 300 juta = **3,06 tahun**

**Kesimpulan**: Proyek financially viable dengan NPV positif dan ROI > 25%.
