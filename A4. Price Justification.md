### 3.2 Justifikasi Spesifikasi

#### **Production Server - Dell PowerEdge R450**

**Rujukan:** KAK Success Criteria - "Aplikasi dapat diakses bersamaan oleh lebih dari satu user"

**Analisis Kebutuhan:**

- **Concurrent Users:** Target 500+ simultan (2.700 LPK dengan peak usage estimation)
- **CPU:** Xeon Silver 4314 (16 cores) untuk handle concurrent requests
- **RAM:** 64GB untuk caching & session management (Laravel + Apache)
- **Storage:** 2x 960GB SSD RAID1 untuk high I/O performance & redundancy
- **Network:** 2x 1Gb NIC untuk redundancy & load balancing

**Vendor Comparison:**

- Dell PowerEdge R450: Rp 45 juta (pilihan kami - balance performa & harga)
- HP ProLiant DL20 Gen10 Plus: Rp 48 juta (lebih mahal)
- Lenovo ThinkSystem SR250: Rp 42 juta (spec lebih rendah)

#### **Database Server - Dell PowerEdge R450**

**Rujukan:** KAK Spesifikasi - "Database SQL Server 2016"

**Analisis Kebutuhan:**

- **RAM:** 128GB untuk SQL Server buffer pool & indexing (2.700+ LPK dengan historical data)
- **Storage:** 4x 1.92TB SSD RAID10 untuk database performance & data protection
- **IOPS:** SSD untuk fast query execution (target < 500ms query time)
- **Network:** 2x 10Gb NIC untuk fast data transfer (migrasi, backup, replication)

**SQL Server 2016 Minimum vs Recommended:**

- Minimum: 4GB RAM, 6GB disk
- Recommended untuk production: 64GB+ RAM, RAID storage
- **Kami:** 128GB RAM, RAID10 SSD (exceed recommendation untuk future-proof)

#### **Backup Storage - Synology NAS**

**Rujukan:** KAK Lampiran II, Point 5 - "Log disimpan setidaknya selama tiga bulan"

**Analisis Kebutuhan:**

- **Capacity:** 4x 4TB = 16TB total, 12TB usable (RAID5)
- **Retention:** Database backup (daily full + hourly incremental) = ~2TB/bulan
- **Audit Logs:** 3-month retention requirement = ~500GB
- **Document Archive:** User uploads, certificates = ~8TB
- **Total Estimated:** ~10TB untuk 6-month retention (safely within 12TB)
